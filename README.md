# Windows 8 Dark Theme for KDE Plasma

A sleek, dark theme **inspired** by the aesthetic of **Windows 8**, crafted for **KDE Plasma 6.5.4**.

## Requirements
- **KDE Plasma 6.5.4** (This theme was created in this version, will probably with future versions. Although I think Klassy requires Plasma 6, but I'm not sure)
- **Klassy** window decoration – [Download here](https://github.com/paulmcauley/klassy)

---

## Configuration

### 1. **Wallpaper**
- File: `Windows_8.jpg`
- Set as desktop background.

### 2. **Color Scheme**
- File: `Windows_8_Dark.colors`
- Apply via **System Settings > Appearance > Colors > Get New > Add from file**.

### 3. **Application Style**
- Select **"Klassy"**.

### 4. **Plasma Style**
- File: `Windows_8_Desktoptheme.tar.gz` 
- Apply via **System Settings > Appearance > Plasma Style > Get New > Add from file**.

### 5. **Window Decorations**
- Choose **"Klassy"** in **System Settings > Appearance > Window Decorations**.
- Apply the following settings in the Klassy configuration panel:

#### **Buttons Tab**
- Preset: **"Defenstrated 8"**
- Icon size: **16**
- Bold icons: **Bold**
- **Button Colors**:
  - Icon colours: **Title bar text colour**
  - Background & outline colours: **Title bar text colour with negative close button**
  - ✅ **Override individual colours**:
    - **✖️ Close**:
      - Icon normal, hover, pressed: `#f0f0f0`
      - Background normal: `#c1544b`

#### **Window Tab**
- **Shadow style**:
  - Strength: **50%**

### 6. **Icons**
- Theme: **"Papirus Dark (Pale orange folders)"**  
  *(Note: Other Windows Vista/7 icon packs were considered but none matched the desired aesthetic.)*

### 7. **Cursors**
- Theme: **"Breeze Dark"** (default)  
  *(Tried alternatives but none were convincing.)*

### 8. **Sounds**
- Theme: **"Free Desktop"**

### 9. **Welcome Screen & SDDM**
- Welcome Screen: **"Breeze"**
- SDDM Theme: **"Breeze"**
    - Wallpaper: `SSDM_Windows_8.jpg`

## Extras
- Profile Picture: `User_Windows_8.png`
- Desktop Layout: See the `Desktop_configuration_example` folder for panel setup ideas.
- Custom Accent Colors:  
  Windows 8 allowed accent color customization. You can adjust:
  - Accent color (default: Windows signature blue)
  - Selected window color (default: creamy-yellow)  
  *Requires manual theming knowledge.*
